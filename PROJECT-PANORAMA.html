项目概况

overview

🎯 项目使命

将 boku（艾玛/小玛）的扁平化技能体系拆分为标准化、可移植的「能力包」，通过统一的适配层部署到不同的 Agent 框架（Hermes / OpenCode / Claude Code）。让每个 Agent 都能复用 boku 积累的能力和经验。

  • 18 个能力领域（包） × ~70 技能簇 × ~200 原子技能
  • 技能 + 经验 + MCP 配置三位一体
  • 三层树状索引 + SQS 质量评分 + 生命周期审计

🏗️ 技术栈

  • 格式: YAML Schema v1 · JSON Schema 验证
  • 核心框架: UCA Core (Protocol + Parser + Verifier)
  • 适配器: HermesAdapter · OpenCodeAdapter
  • 质量体系: SQS 五维评分 · SQLite 持久化 · 历史趋势
  • CI/CD: GitHub Actions · 4 job 并行门禁
  • 运行时集成: SRA v1.4.0 质量加权推荐

  • merge-suggest — 技能合并建议引擎
