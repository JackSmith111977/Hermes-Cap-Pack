## 用户故事

**As a** 主人
**I want** 一个可视化的健康度仪表盘，展示 skill 系统随时间变化的质量趋势
**So that** 一目了然地看到系统是在变好还是变差，哪些领域需要关注

## 验收标准

- [ ] AC-01: HTML 仪表盘自动从 SQS 数据库生成
- [ ] AC-02: 包含历史趋势折线图（过去 4-8 周的 SQS 均线）
- [ ] AC-03: 包含 18 个包的雷达图对比
- [ ] AC-04: 包含低分 skill 排行榜
- [ ] AC-05: 包含退化/晋升图示
- [ ] AC-06: 可通过 cron 每周自动更新并发送

## 技术方案

1. 基于 `skill-quality.db` 的历史数据生成 HTML 页面
2. 使用 Chart.js (CDN) 渲染图表
3. 设计暗色主题（与现有 reports/ 风格一致）
4. 纳入 `reports/health-dashboard.html`
5. cron 任务在生成报告的同时更新仪表盘
