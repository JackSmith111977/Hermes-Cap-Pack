# pdf-layout

> **源 Skill**: `~/.hermes/skills/pdf-layout/SKILL.md`
> **版本**: 来自 Hermes 技能库
> **用途**: 使用 ReportLab 和 WeasyPrint 生成专业 PDF 文档

## 来源

此技能的定义存储在 Hermes Agent 技能库中。能力包引用的是路径映射，而非复制内容。

**映射**: `~/.hermes/skills/pdf-layout/SKILL.md`

## 关联经验

- `wqy-font-fallback` — WQY 字体回退
- `reportlab-cjk-encoding` — 中文字体编码

## 安装提示

```bash
pip install weasyprint reportlab
# Ubuntu: apt install libpango-1.0-0 libpangocairo-1.0-0 libgdk-pixbuf2.0-0
```
