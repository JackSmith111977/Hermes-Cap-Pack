# Learning Cycle

## Definition

The Learning Cycle is a structured methodology for acquiring knowledge and skills in any domain. It moves through distinct phases — breadth-first survey, deep-dive iteration, skill precipitation, and review loops — forming a continuous improvement spiral. Unlike ad-hoc learning (reading randomly, watching tutorials without practice), the learning cycle ensures efficient progression from novice to competent practitioner.

## Core Concepts

### The Four-Phase Cycle

```
Phase 1: SURVEY (Breadth-first)
  → Map the domain, identify key concepts and terminology
  → Create a knowledge graph of sub-areas
  → Time-box to avoid analysis paralysis

Phase 2: DIVE (Deep iteration)
  → Pick one sub-area and go deep enough to practice
  → Follow the 3-iteration rule: reproduce → modify → create
  → Use active learning (practice, not just read)

Phase 3: PRECIPITATE (Skill extraction)
  → Extract patterns and procedures from experience
  → Write Experience documents, create reusable Skills
  → Connect new knowledge to existing mental models

Phase 4: REVIEW (Reinforcement)
  → Spaced repetition at increasing intervals
  → Active recall (retrieve from memory, not re-read)
  → Cross-link knowledge across domains
```

### Learning Depth Levels

| Level | Behavior | Evidence |
|:------|:---------|:---------|
| **L1: Awareness** | Knows the term exists | Can recognize it in context |
| **L2: Comprehension** | Understands the concept | Can explain it in own words |
| **L3: Application** | Can use it with guidance | Follows tutorials successfully |
| **L4: Proficiency** | Can use it independently | Solves novel problems alone |
| **L5: Mastery** | Can teach and innovate | Creates new patterns, teaches others |

### Active vs Passive Learning

Passive (10-20% retention): Reading, listening, watching
Active (50-90% retention): Practice, discussion, teaching, applying

The learning cycle emphasizes active phases (Dive, Precipitate) over passive consumption (Survey is deliberately time-boxed).
